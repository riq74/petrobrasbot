## Instalar

```bash
> termux-setup-storage
(depois disso toque na permissão)
> apt install git
> pkg install ffmpeg
> apt install wget
> pkg install nodejs
> pkg install npm
> git clone https://github.com/riq74/petrobot2.git
> cd petrobot
> bash install.sh
```

## Uso

```bash
> npm start
> leia o códico QR e após isso de um exit
> cd petrobot
> npm i -g pm2
> pm2 start index.js
> pm2 monit
```
